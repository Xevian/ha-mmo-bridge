╔══════════════════════════════════════════════╗
║         MMO Hub — Setup Guide               ║
╚══════════════════════════════════════════════╝

The MMO Hub connects your Second Life presence
to Home Assistant so the owner can see when
you're online, send you messages in-world, and
run automations based on your status.

───────────────────────────────────────────────
  STEP 1 — Rez the MMO Hub
───────────────────────────────────────────────

Rez the MMO Hub object on your home parcel. It
should stay rezzed permanently so Home Assistant
always has an active connection.

Make sure the MMO Hub's group is set to the
group you want to use for registration — only
members with that group tag active can register.

The MMO Hub will rename itself to reflect the
parcel it's on (e.g. "MMO Hub - Xev Getaway").

───────────────────────────────────────────────
  STEP 2 — Register
───────────────────────────────────────────────

Make sure your group tag is active, then TOUCH
the MMO Hub.

You'll receive an IM confirming registration.
That's it — you're connected!

If you're already registered and touch again,
your HUD URL will be refreshed automatically.

───────────────────────────────────────────────
  WHAT GETS TRACKED
───────────────────────────────────────────────

  • Online / offline status
  • Whether you're at the home parcel
  • Region and parcel name (via HUD)
  • AFK and busy flags (via HUD)

───────────────────────────────────────────────
  RECEIVING MESSAGES
───────────────────────────────────────────────

The owner can send you IMs from Home Assistant
automations. They arrive as normal IMs from
the MMO Hub and look like:

  ====================
  == Dinner's ready!
  ====================

───────────────────────────────────────────────
  OWNER COMMANDS  (channel /5)
───────────────────────────────────────────────

  /5 seturl <url>    Save HA webhook URL
  /5 setpoll <sec>   Presence poll interval
                     (min 10s, default 60s)
  /5 status          HA URL, script URL,
                     registered avatars
  /5 list            List registered avatars
  /5 remove <name>   Remove an avatar by name
  /5 clearusers      Remove all avatars
  /5 push            Force a presence update
  /5 hardreset       Clear all stored data
                     (use when moving to new HA)
  /5 help            Show this list in chat

───────────────────────────────────────────────
  MMO NODE — PARCEL STATS
───────────────────────────────────────────────

The MMO Node is a separate optional object that
pushes live parcel and region stats to Home
Assistant as graphable sensors:

  • Region FPS and time dilation
  • Number of avatars on the parcel
  • Number of avatars in the region
  • Sim channel, version, and start time

Rez the MMO Node somewhere on the parcel that
stays rezzed permanently (a skybox or ground
platform works well). Set it up the same way
as the MMO Hub:

  /5 seturl <your webhook URL>

It registers with HA automatically and starts
pushing stats on its own poll cycle. The MMO
Node and MMO Hub are independent — you can have
one without the other.

───────────────────────────────────────────────
  NEED TO UNREGISTER?
───────────────────────────────────────────────

Ask the owner to run:  /5 remove Your Name

───────────────────────────────────────────────
  PROBLEMS?
───────────────────────────────────────────────

Not receiving IMs? Make sure you registered by
touching the MMO Hub with your group tag active.

Contact the owner if you continue having
trouble.
